# Перенос базы PocketBase: план

## Цель
Перенести локальную `pb_data/` с исходного компьютера на этот.

## Шаги

### 1. Подготовка (на обоих компьютерах)
- [ ] Закрыть браузер с открытым PB (http://127.0.0.1:8090)
- [ ] Остановить PocketBase, если запущен:
  ```bash
  lsof -i :8090
  kill <PID>
  ```

### 2. Копирование данных
Выбрать один способ:

**Вариант A — SSH (если компьютеры в одной сети)**
```bash
# на этом компьютере:
rsync -avz user@192.168.x.x:/путь/до/Table/pb_data/ ./pb_data/
```

**Вариант B — USB-флешка**
- На исходном ПК: скопировать папку `pb_data/` на флешку
- На этом ПК: скопировать с флешки **поверх** `pb_data/`

### 3. Запуск и проверка
- [ ] Запустить PocketBase:
  ```bash
  pnpm pb
  ```
- [ ] Открыть `http://127.0.0.1:8090/_/` и проверить данные
- [ ] Открыть `http://127.0.0.1:5173` и проверить, что приложение работает

### 4. Если что-то пошло не так
- PocketBase не стартует — удалить `pb_data/` и запустить `pnpm pb`, чтобы создать чистую БЗ, затем миграции накатятся автоматически
- Нужен пароль суперпользователя — сбросить через терминал:
  ```bash
  ./pocketbase.exe superuser upsert email@example.com newpassword
  ```
  (заменить email@example.com и newpassword)
